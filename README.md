# Django Word Counter

A simple Django application that counts the number of words in a paragraph.

## How to use

1. Clone this repository:
```bash
git clone https://github.com/ashish5507/django-word-counter.git
cd django-word-counter
```

2. Create a virtual environment:
```bash
python3 -m venv venv
source venv/bin/activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Run the server:
```bash
python manage.py runserver
```

5. Open your browser and go to `http://127.0.0.1:8000/`

## Features

- Enter any paragraph
- Get the word count instantly
- Simple and clean UI

- Django
